Project description

The project involves the development of an application for monitoring the production and storage of wine in a vineyard.
In particular, humidity and temperature values for production are monitored, and motion sensors are implemented both in production and storage, to detect possible intrusions.
The software is developed using Arduino, and uploaded to ESP8266 cards with internet connection.
Sensor and alarm data are read by the microcontroller and sent to an SQL database via the MQTT protocol in json format.
The values are then displayed on a web page, with backend in Python language.
Alarms are notified via a telegram bot, which signals the user if the temperature in the storage is above a threshold, or if an intrusion is detected.
