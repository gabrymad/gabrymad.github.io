Image Auto Orientation

University project

Course: Advanced Machine Learning

Date: 31/01/2022

Final score: 30 / 30

Project description

The objective of the project is to classify the rotations of images inherent in certain scenarios, including landscapes, cities, roads, etc., and to make any rotation corrections, through the use of deep learning models.
Rotations are allowed at 0, 90, 180 and 270 degrees.
The scenarios considered are as follows:

  • Buildings
  • Streets
  • Mountains
  • Sea
  • Forest
  • Glacier

For each of these scenarios, a Convolutional Neural Network with the same architecture is implemented and trained.
In addition, a seventh CNN is implemented, which is trained with all six scenarios.
The models are developed using the Python Keras framework.
The following metrics were used to assess the capabilities of the models: precision, recall and f1-score.
Furthermore, a more in-depth study of the network's behavior during the prediction of rotations was conducted, going to display the portions of the image that most contributed to its classification in the 4 possible rotations introduced earlier.
