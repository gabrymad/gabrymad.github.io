Image Auto Orientation
Computer Vision üíª
University Project

A Deep Learning model is trained from scratch to predict and correct rotation on various contestualized images, such as buildings, mountains, sea, streets, etc...
Rotations can be 0, 90, 180 and 270 degrees.
Models are evaluated using F1-score, precision, recall. Furthermore, a more in-depth study of network behaviour is conducted to display the image's portions that most contributed to its classification.

Computer Vision üíª
University Project

A classifier is built to analyze and localize irregularities in chocolates inside a box.
No deep learning applied, the classifier performs by defined rules.

The anomalous chocolates are highlighted with a circle in the input image.
